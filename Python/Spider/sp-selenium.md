# Selenium
---
>selenium是一个用于Web应用程序测试的工具
selenium可以直接操作浏览器，就像真正的用户一样
selenium支持多种编程语言（JAVA\python\js\c#\ruby），进行测试脚本的编写
selenium支持在多个浏览器平台上执行测试
selenium是一个工具集，有了这个工具集，就可以顺利开展自动化测试
